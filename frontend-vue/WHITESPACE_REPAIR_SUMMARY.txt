常见问题快速解答
================================================================================

Q: 为什么修改代码后出现白屏？
A: 因为添加了新依赖 (@vueuse/core)，需要运行 npm install

Q: 安装后还是白屏怎么办？
A: 清除浏览器缓存：Ctrl+Shift+R 硬刷新

Q: npm install 总是失败？
A: 试试用国内镜像：
   npm install --registry https://registry.npmmirror.com

Q: 如何快速排查问题？
A: 按照这个顺序：
   1. npm install
   2. Ctrl+Shift+R
   3. npm run dev
   4. 打开 F12 查看 Console 错误
