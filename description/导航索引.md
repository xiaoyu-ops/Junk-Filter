## 启动系统（快速参考）

**VS Code 一键启动（推荐）**：
```
Cmd + Shift + B  →  🚀 Start All Services
```

服务地址：
- 前端: http://localhost:5173
- Go API: http://localhost:8080/health
- Python API: http://localhost:8083/health
